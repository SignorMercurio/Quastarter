# Quastarter

Opinionated Quasar v2 Starter Template

## Install the dependencies

```bash
yarn
```

## Generate App Icons

```bash
./genIcon.sh
```

## Start the app in development mode (hot-code reloading, error reporting, etc.)

```bash
npm run web:dev
```

## Lint the files

```bash
yarn run lint
```

## Build the app for production

```bash
npm run web:prod
```
